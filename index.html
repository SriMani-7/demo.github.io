Project setup in 420 Seconds:

  1. Create a Maven project (name is your choice)
  2. Create a Git repository for the project
  3. Add a .gitignore file
  4. Make the initial commit
  5. Create a new repository on your GitHub account
  6. Set the GitHub repository as the origin for the local repo
  7. Push local changes to the origin
